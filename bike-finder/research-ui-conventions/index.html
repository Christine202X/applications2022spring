Bose - a white noise cancelling headphones

What UI elements did I see and/or interact with while finding, comparing, selecting, and configuring the specific product?

The navigation bar, the dropdown menu, the comparison feature, the color options, the buy button, the reviews, the buy now button.

Have I seen this UI element before in other contexts?

I have seen most of them except the comparison feature which I think is interesting because it allows the user to choose what products you want to compare instead of listing everything on the page.

Did I know what this element would do before I interacted with it? Why?

Yes, because the elements are straightforward: using text instead of icons.

What general problem is this element designed to handle?

The comparison feature is designed because there are a lot of similar products within the noise cancelling category and it makes sense to design an element like this for users to compare easily.

Why was it used in this context? Is it successful?

It is used because users probably don't have the patience to go through every page by themselves, so I think this function is successful in terms of helping users save time.

Macy's - a blue casual shirt

What UI elements did I see and/or interact with while finding, comparing, selecting, and configuring the specific product?

The navigation bar, the dropdown menu, a filter of different category of shirts, a filter feature on the left with other options such as color, a “more like this” dropdown to compare similar shirts, options for color and size(ui including “size chart” and “what size fit me”), an add to bag button

Have I seen this UI element before in other contexts?

The filter: I’ve seen it elsewhere but here the designer separates the filter for different types of shirt(casual, t-shirt, dress, polo) and the filter for other options such as sales/brand/size/color/price, etc.

Did I know what this element would do before I interacted with it? Why?

I did because they have checkboxes and round buttons that look clickable.

What general problem is this element designed to handle?

To handle the problem that there are many different categories of shirts as well as other characteristics.

Why was it used in this context? Is it successful?

To help users categorize and find the shirt that they are looking for.

Summary

What elements or patterns seem universal?

Navigation bar, dropdown menu, filter option, different categories(color, size)

Did I see anything that felt different or unusual?

Yes, it felt unusual and unnecessary to have “shop by category”, “shop by Metal”, and “shop by Gemstone” option on the dropdown menu when some of those options could be combined or made as filters.

What elements or patterns were specific to the store or product?

Some compare elements as well as the filters are specific to different stores.

Were there any patterns or techniques that made one store stand out? Could the other stores have applied that technique successfully?

The filter option for Macy’s and the comparison feature for Bose. I think other stores could also have a more detailed but straightforward filter like Macy’s when there are lots of products.

Finally, how did you feel when interacting with these sites? When were you happiest/most comfortable? When were you disrupted/confused/bothered?

I was most comfortable with the filter for Macy’s because it has many different options for me. I felt confused when I wanted to look for a rose gold necklace but there are both a “necklace” category and a “rose gold” category.
